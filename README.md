# GNOME-CN计划
## gnome中文翻译包(早于官方版)
### GNOME官方翻译人员凋敝，进度缓慢，于是将自用的翻译包发到本仓库供大家使用，也欢迎对翻译的修改和提议。

## Version
#### GNOME 3.36

## 用法
`sudo bash ./install.sh` 安装

`sudo bash ./install.sh r` 恢复
